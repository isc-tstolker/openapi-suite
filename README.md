# openapi-suite

## Installation ZPM

In an IRIS Terminal

```
zpm "install openapi-suite"

; optional
zpm "install swagger-ui"
```

## Installation docker

```
git clone git@github.com:lscalese/openapi-suite.git
cd openapi-suite
docker-compose up -d
```
